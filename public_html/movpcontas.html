Pagar Conta


A tela de Pagar Conta tem como objetivo de registrar as contas que foram pagas.


Campos obrigatórios

Caixa, Data pagamento e Valor pago.


Campos não obrigatórios

Local.


Para Alterar:

Para alterar uma conta, selecione o registro na tabela e clique no botão Alterar ou clique com dois cliques sobre o registro. Após o preencher os campos, clique no botão Gravar para salvar as alterações.


Para Estornar:

Para estornar uma conta já paga, clique no botão Estornar, busque um registro e selecione o registro com dois cliques para ser Estornado. Agiuarde um momento que será exibido uma mensagem informando que foiestornado.
