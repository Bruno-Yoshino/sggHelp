Gerenciamento de Parcelas


A tela de Gerenciamento de parcelas tem como objetivo de gerar parcelas referentes a Compra, Despesa ou Conta a receber de um cliente.


Para Gerar Parcelas

Após ser redirecionado para a Tela de Gerenciamento de Parcelas, o usuário informa a quantidade de parcelas e o intervalo das parcelas.

Os tipos de intervalos são:

0 para 1 mes.

1 ~ 31 para Dia(s)

-1 ~ -3 para Semana(s)


Para Alterar uma parcela

Selecione um registro e clique no botão Alterar o valor da parcela. Após a alteração, crique em Atualizar.


Para Gravar

Clique no botão Gravar.
