Despesa


A tela de despesa tem como objetivo de lançar despesas geradas na empresa.


Campos obrigatórios

Tipo, Valor do documento, Data vencimento e Pagamento.


Campos não obrigatórios

Nome da conta e Código de Barras.


Para Incluir:

Para registrar uma nova despesa, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar uma nova despesa, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir uma nova despesa, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
