Relatório de Comprovante de pagamento


O Comprovante de pagamento tem como objetivo de fazer papel de um comprovante após o recebimento de um cliente, ou seja, um comprovante de pagamento.


Opções

Número conta.


Para Gerar (Após clicar no botão Gravar):

Após clicar no botão Gravar da Tela de Receber Conta, aparecera uma mensagem perguntando se deseja gerar um comprovante. O usuário clica na opção Sim, escolha o lugar para armasenar, dê um nome para o arquivo e clique em Gravar.


Para Gerar (Pelo botão Imprimir):

Após clicar no botão Imprimir, será carregado uma tela de consulta. Clique em localizar e selecione o registro com dois cliques. Após ter selecionado, aparecera uma tela para realizar o armazenamento, especifique o locar para armazenar, dê um nome para o arquivo e clique em Gravar.


PDF: O relatório será gerado em formato .pdf, nessa opção o usuário informa o local a ser salvo e o nome do arquivo.
