Orçamento


A tela de cheque tem como objetivo de lançar cheque(s) recebido(s) de um cliente e gerenciar se o cheque foi compençado ou não.


Campos obrigatórios (Dados do cheque)

Valor, Pré datado, Nome da pessoa do cheque, CPF do Dono do Cheque, Número da Agência, Número da Conta, Número do Banco e Número do Cheque.


Campos obrigatórios (Para compensar)

Data compensado.


Campos não obrigatórios

Observação, Nome do Cliente.


Para Incluir:

Para registrar um novo Cheque, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um Cheque, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um Cheque, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.


Para Compensar cheque:

Para compensar um Cheque, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Compensar. Informe a data compensado, se foi compensado e seleciona o motivo. Após o término, clique no botão Gravar para compensar o cheque.
