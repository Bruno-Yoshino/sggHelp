Cadastro de Cliente


A tela de Cadastro de Cliente tem como objetivo de cadastrar clientes de pessoa fisica ou jurídica para poder realizar Orçamento ou Pedido de serviço(s).


Campos obrigatorios (Dados do Clietne)

Nome/Firma, Estado, Cidade, Endereço, Número, CEP, Telefone e/ou Celular.

Campos obrigatorios (Pessoa Fisica)

CPF, RG, Org. Emissor e Data de nascimento.

Campos obrigatorios (Pessoa Juridica)

Razão social, CNPJ e Inscrição Estadual.

Campos não obrigatorios(Dados do Clietne)

Complemento e e-mail.


Para Incluir:

Para incluir um novo registro, clique no botão Novo com o botão direito do mause, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um registro, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o termino, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um registro, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para excluir o registro pesquisado.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuario podera estar selecionando opções e filtros para a busca.
