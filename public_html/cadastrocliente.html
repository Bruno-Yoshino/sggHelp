Cadastro de Cliente


A tela de Cadastro de Cliente tem como objetivo de cadastrar clientes de pessoa fisica ou jurídica para poder realizar Orçamento(s) ou Pedido(s) de serviço(s).


Campos obrigatórios (Dados do Cliente)

Nome/Firma, Estado, Cidade, Endereço, Número, CEP, Telefone e/ou Celular.

Campos obrigatorios (Pessoa Física)

CPF, RG, Org. Emissor e Data de nascimento.

Campos obrigatorios (Pessoa Jurídica)

Razão social, CNPJ e Inscrição Estadual.

Campos não obrigatórios(Dados do Cliente)

Complemento e e-mail.


Para Incluir:

Para incluir um novo Cliente, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um Cliente, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um Cliente, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
