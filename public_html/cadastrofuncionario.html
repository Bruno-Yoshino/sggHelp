Cadastro de Funcionario


A tela de Cadastro de Funcionário tem como objetivo de cadastrar funcionários para poder realizar serviços de dia a dia, tais como: Orçamento, Pedido, Ajuste de Estoque (Produto/Folha), Produção, Compra, Receber Conta e Pagar Conta.


Campos obrigatórios

Nome/Firma, Estado, Cidade, Endereço, Número, CEP, Telefone e/ou Celular.

Campos não obrigatórios

Complemento e Foto.


Para Incluir:

Para incluir um novo Funcionário, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um Cliente, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um Funcionário, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
