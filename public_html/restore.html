Restore (Restaurar)

Caso ja esteja logado, pure até a etapa 5.

1- Executar o aplicativo PgAdmin 4.

2- Ao executar, clique no + do Servers.


3- Clique no + do PostgresSQL 10.


4- Informe a senha criada na hora de instalar o Postgres. Neste caso informar: postgres123

Caso queira pode deixar checado a opção Save Password para deixar a senha salvo.


5- Clique com o botão direito do mouse sobre o sgg e seleciona a opção Restore... .

Ou

5- Clique em Tools e clique na opção Restore.


6- Clique no botão do File ... conforme a imagem acima.


7- Selecione o arquivo e clique no Botão Select. Caso não encontre o arquivo, troque o Format: para All Files.


8- Após ter clicado no Select, clique no botão Restore.

Observação: Pode levar um tempo.


9- Após ter clicado no Restore, se aparecer uma mensagem "Sucefull" no canto inferior direito, o Resore foi feito com sucesso.
