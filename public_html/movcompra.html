Lançar Compra


A tela de Lançar compra tem como objetivo de lançar compras realizadas pela empresa. Podera estar lançando compra de produto(s) e/ou folha(s).


Campos obrigatórios (Para Produto)

Produto, Quantidade e Preço Unitario.


Campos obrigatórios (Para Folha)

Folha/Tamanho, Quantidade e Preço Unitario.


Campos não obrigatórios

Fornecedor.


Para Incluir:

Para registrar uma nova compra, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar uma compra, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir uma compra, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
