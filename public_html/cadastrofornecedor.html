Cadastro de Fornecedor


A tela de Cadastro de Fornecedor tem como objetivo de cadastrar fornecedores para poder realizar Compras ou somente para manter as informações do fornecedor.


Campos obrigatórios

Nome, Estado, Cidade e Telefone e/ou Celular.

Campos não obrigatórios(Dados do Cliente)

endereço e e-mail.


Para Incluir:

Para incluir um novo Fornecedor, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um Fornecedor, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um Fornecedor, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
