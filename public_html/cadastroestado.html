Cadastro de Estados


A tela de Cadastro de Estado tem como objetivo de cadastrar estado para poder selecionar o estado em Cadastro de Cidade.


Campos obrigatórios

Sigla.


Para Incluir:

Para incluir um novo Estado, clique no botão Novo com o botão direito do mouse, preencha o campo e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um Estado, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um Estado, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
