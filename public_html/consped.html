Consulta de Pedido


A tela de Consulta de Pedido tem como objetivo de consultar pedidos cadastrados.


Opções

Tudo, Data, Periodo, Número e Nome (Cliente)


Campo não obrigatório

Valor e Data.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas.


Para Visualizar os Serviços:

Selecionar o registro do Pedido. O sistema automaticamente carrega o(s) Serviçõ(s) na Tabela que está a Esquerda.


Para Visualizar os Detalhes do Serviços:

Selecionar o registro do Serviço. O sistema automaticamente carrega o(s) Detalhe(s) na Tabela que está a Diretita.
