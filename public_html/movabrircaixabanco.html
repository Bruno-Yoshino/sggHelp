Abrir Caixa Banco


A tela de abrir caixa banco tem como objetivo de estar cadastrando a conta do banco e o seu saldo para poder estar controlando os pagemntos.


Campos obrigatórios

Nome da caixa e Saldo.


Para Inserir:

Clique no botão novo para habilitar os campos e preencha os campos obrigatórios. Após o preenchimento, clique no botão gravar para salvar os registros.


Para Alterar:

Para alterar uma Caixa, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir uma Caixa, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
