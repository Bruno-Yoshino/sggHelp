Para Incluir:

Para registrar um novo orçamento, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um Orçamento, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um Orçamento, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.


Para Gerar PDF:

Para Gerar PDF, busque um registro através do botão Localizar e selecione o registro com dois cliques. Após o retorno, clique no botão Gerar PDF, selecione o local par salvar, nome do arquivo e clique em OK.
