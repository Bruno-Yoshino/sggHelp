Ajustar Caixa Banco


A tela de ajustar caixa banco tem como objetivo de adicionar ou retirar (creditar ou debitar) dinheiro de uma caixa de um banco cadastrado.


Campos obrigatórios

Saldo a ser Debitado/Creditado e o Tipo (Debitar ou Creditar).


Para Ajustar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca. Clicar duas vezes sobre o registro para retornar. Após o retorno, o sistema automaticamente libera os campos, preencha os campos e clique no botão Gravar para alterar registro.
