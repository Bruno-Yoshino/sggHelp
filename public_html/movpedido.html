Pedido


A tela de pedido tem como objetivo de registrar um pedido de serviço solicitado pelo cliente.


Campos obrigatórios (Para pedido)

Nome do Cliente.


Campos obrigatórios (Para serviço)

Serviço, Quantidade e Valor/unitário.


Campos obrigatórios (Para detalhe)

Descrição do Serviço.

Dependendo da descrição selecionado: Numeração, Vias ou Outros.


Campos obrigatórios (Para entrega)

Forma de pagamento.


Campos não obrigatórios

Desconto, Valor Extra e Descrição.


Para Incluir:

Para registrar um novo pedido, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar um pedido, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Entregar (Dar baixa):

Para dar baixa em um pedido, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Entregar. Após o selecionar a forma de pagamento, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir um pedido, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.


Para Gerar PDF:

Para Gerar PDF, busque um registro através do botão Localizar e selecione o registro com dois cliques. Após o retorno, clique no botão Gerar PDF, selecione o local par salvar, nome do arquivo e clique em OK.
