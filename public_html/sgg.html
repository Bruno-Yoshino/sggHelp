Passos para criar o Banco

Caso tenha feito essa primeira parte, pure para a segunda parte: Passos iniciar o sistema

Executar o aplicativo PgAdmin 4.

Ao executar, clique no + do Servers.


Clique no + do PostgresSQL 10.


Informe a senha criada na hora de instalar o Postgres. Neste caso informar: postgres123

Caso queira pode deixar checado a opção Save Password para deixar a senha salvo.


Com o botão diretiro diretor do mouse, clique em cima do databases, selecione a opção Create e clica na opção Database.


No campo database, informe conforme a imagem acima: sgg. Clique em Save


Clique no + do database, + do sgg, clique em Tools e clique na opção Query Tool.


Clique no + do database, + do sgg, clique em Tools e clique na opção Query Tool.


Abra o arquivo Create em bloco de notas que esta dentro da pasta do projeto, copia e cola no Query conforme a imagem.

Após ter feito, clique na opção Execute(esta com icone de Raio).


Realize a mesma operação para o arquivo Trigger.txt e Function VerificaEstoque.txt


Pronto. Agora está preparado para executar o sistema.


Passos iniciar o sistema

Dentro da pasta, terá um arquivo executavel com nome SGG (conforme a imagem acima).


Clica duas vezes para executar o sistema.


Pronto, o sistema já esta sendo executado.
