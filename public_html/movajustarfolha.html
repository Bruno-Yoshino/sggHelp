Ajustar Folha


A tela de ajustar folha tem como objetivo de ajustar o estoque de folha. Nesta tela, o usuário posera estar Adicionando ou Retirando Folha.


Campos obrigatórios

Serviço, Folha, Quantidade a ser retirado e Observação.


Para ajustar

Clique no botão novo para liberar os campos para ser preenchidos. Após o preenchimento, clique no botão Gravar para slavar o ajuste.
