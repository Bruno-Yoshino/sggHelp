Cadastro de Folha


A tela de Cadastro de Folha tem como objetivo de cadastrar folhas para poder ser utilizados em Ajustar Estoque de Folha, Compra e Produção.


Campos obrigatórios

Tamanho, Descrição, Altura e Largura.

Campos não obrigatórios

Ativo.


Para Incluir:

Para incluir uma nova Folha, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar uma Folha, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir uma Folha, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
