Cadastro de Produto


A tela de Cadastro de Produto tem como objetivo de cadastrar cidades para poder ser utilizados em Cadastro de Funcionarios, Cadastro de Cliente, Cadastro da Empresa e Cadastro de Fornecedor.


Campos obrigatórios

Nome e Estado.


Para Incluir:

Para incluir uma nova Produto, clique no botão Novo com o botão direito do mouse, preencha os campos e clique no botão Gravar para gravar o novo registro.


Para Alterar:

Para alterar uma Produto, busque um registro através do botão Localizar, selecione o registro com dois cliques e clique no botão Alterar. Após o término, clique no botão Gravar para salvar as alterações.


Para Excluir:

Para excluir uma Produto, busque um registro através do botão Localizar e selecione o registro com dois cliques para ser excluido. Após o retorno, clique no botão Excluir para efetivar a operação.


Para Pesquisar:

Clique no botão Localizar para realizar pesquisas, aqui o usuário poderá estar selecionando opções e filtros para a busca.
