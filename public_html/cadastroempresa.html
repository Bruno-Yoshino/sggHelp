Cadastro de Empresa


A tela de Cadastro de Empresa tem como objetivo de cadastrar os dados da empresa para ser utilizados em relatorios.


Campos obrigatórios

Nome, CNPJ, Estado, Cidade, Endereço, Número, CEP, Telefone e e-mail.

Campos não obrigatórios

Imagem.


Para Incluir:

Ao carregar a tela de cadastro, o sistema automaticamente libera os componentes para ser preenchidos. Após o preenchimento dos campos, clique no botão Gravar para gravar o registro.


Para Alterar:

Ao carregar a tela de cadastro, o sistema busca os dados da empresa automaticamente e libera para realizar alteração. Após o término, clique no botão Gravar para salvar as alterações.


Para Pesquisar:

O sistema automaticamente busca os dados.
